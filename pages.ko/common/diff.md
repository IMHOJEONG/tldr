# diff

> 파일과 디렉터리를 비교.
> 관련 항목: `delta`, `difft`.
> 더 많은 정보: <https://manned.org/diff>.

- 파일을 비교 (`과거_파일`을 `새로운_파일`로 변경하기 위한 차이를 표시):

`diff {{경로/대상/과거_파일}} {{경로/대상/새로운_파일}}`

- 공백을 무시하고, 파일을 비교:

`diff {{[-w|--ignore-all-space]}} {{경로/대상/과거_파일}} {{경로/대상/새로운_파일}}`

- 파일을 나란히 비교하여, 차이를 표시:

`diff {{[-y|--side-by-side]}} {{경로/대상/과거_파일}} {{경로/대상/새로운_파일}}`

- unified 형식(`git diff`)으로 차이를 표시:

`diff {{[-u|--unified]}} {{경로/대상/과거_파일}} {{경로/대상/새로운_파일}}`

- 디렉터리를 재귀적으로 비교 (서로 다른 파일/디렉터리 이름과 파일의 변경 내용을 표시):

`diff {{[-r|--recursive]}} {{경로/대상/과거_디렉토리}} {{경로/대상/새로운_디렉토리}}`

- 디렉터리를 비교하고, 서로 다른 파일 이름만 표시:

`diff {{[-r|--recursive]}} {{[-q|--brief]}} {{경로/대상/과거_디렉토리}} {{경로/대상/새로운_디렉토리}}`

- 두 텍스트 파일의 차이로 Git용 패치 파일을 생성(존재하지 않는 파일은 빈 파일로 간주):

`diff {{[-a|--text]}} {{[-u|--unified]}} {{[-N|--new-file]}} {{경로/대상/과거_파일}} {{경로/대상/새로운_파일}} > {{경로/대상/차이.patch}}`

- 색상으로 출력하며, 최소한 변경 집합을 찾도록 파일을 비교:

`diff {{[-d|--minimal]}} --color=always {{경로/대상/과거_파일}} {{경로/대상/새로운_파일}}`
