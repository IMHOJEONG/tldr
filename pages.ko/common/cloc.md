# cloc

> 코드 줄 수를 계산.
> 더 많은 정보: <https://github.com/AlDanial/cloc#options->.

- 디렉터리의 모든 코드 줄 수를 계산:

`cloc {{경로/대상/디렉터리}}`

- 두 디렉터리 구조를 비교해 코드 줄 수의 차이를 계산:

`cloc --diff {{경로/대상/디렉터리1}} {{경로/대상/디렉터리2}}`

- `.gitignore`에 지정된 파일과 같이 VCS에서 무시되는 파일을 제외:

`cloc --vcs git {{경로/대상/디렉터리}}`

- 언어 별이 아닌 파일별로 결과를 표시:

`cloc --by-file {{경로/대상/디렉터리}}`
