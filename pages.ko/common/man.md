# man

> 매뉴얼 페이지 형식화 및 표시.
> 관련 항목: `whatis`, `apropos`.
> 더 많은 정보: <https://manned.org/man>.

- 명령어의 메뉴얼 페이지 표시:

`man {{명령어}}`

- 브라우저에서 명령어 메뉴얼 페이지 열기 (`$BROWSER` 설정 시 브라우저 이름 생략 가능):

`man {{[-H|--html=]}}{{브라우저_이름}} {{명령어}}`

- 특정 섹션(예: 섹션 7)의 메뉴얼 페이지 표시:

`man 7 {{명령어}}`

- 명령어에 대해 사용 가능한 모든 섹션 조회:

`man {{[-f|--whatis]}} {{명령어}}`

- 메뉴얼 페이지 검색 경로 표시:

`man {{[-w|--path]}}`

- 메뉴얼 페이지 내용을 보여주지 않고 위치만 표시:

`man {{[-w|--where]}} {{명령어}}`

- 특정 로케일로 메뉴얼 페이지 검색:

`man {{[-L|--locale]}} {{로케일}} {{명령어}}`

- 특정 문자열이 포함된 메뉴얼 페이지 검색:

`man {{[-k|--apropos]}} "{{검색_문자열}}"`
